# Flask App Setup

# Open terminal in project directory

# Create virtual environment
```bash
python -m venv venv
```

# Activate virtual environment
# On Windows
```bash
venv\Scripts\activate
```
# On macOS/Linux
```bash
source venv/bin/activate
```

# Install requirements
```bash
pip install -r requirements.txt
```

# Run the app
```bash
python app.py
```
